# Vision — itemsforsale.in

itemsforsale.in is a personal selling board where one seller lists household items and receives direct enquiries.

This is NOT a marketplace.
This is NOT multi-seller.
This is NOT an e-commerce checkout flow.

It is a focused website for one owner selling personal belongings.

Core goals:

- List items with photos, pricing context, and status
- Let visitors browse in grid or table view
- Capture buyer interest and direct seller contact submissions
- Protect contact access with a lightweight human captcha
- Manage inventory and enquiries from an admin panel
- Export data as CSV for sharing and offline tracking

Target use case:

- Selling household items locally
- Small-batch personal liquidation
- No payment gateway integration

Principles:

- Simple and readable UX
- Fast and local-first by default
- Low operational cost
- Vercel-ready deployment shape
- Supabase-ready data model
- Easy to maintain and extend
