# Vision — itemsforsale.in

itemsforsale.in is a personal item selling website where a single seller can list household items for sale.

This is NOT a marketplace.
This is NOT OLX.
This is NOT multi-seller.

It is a personal selling board for one owner.

Goal:

- List items with photos and details
- Allow visitors to submit interest / bid
- Collect contact details
- Manage items from admin panel

Target use case:

- Selling household items
- Selling personal belongings
- Small scale selling
- No payment integration

Principles:

- Simple
- Clean
- Fast
- Low cost
- Vercel friendly
- Supabase friendly
- Easy to maintain
