# MVP Scope

Included

- item listing
- item listing grid/table toggle
- item detail page
- interest form
- contact seller page with captcha
- admin login
- admin item CRUD
- admin photo upload
- admin leads viewer
- admin leads filtering and CSV export
- admin contact submissions viewer
- admin system status page (data mode + database health)
- status badge
- search
- filter
- seller location and map embed
- catalogue CSV export with item links
- leads CSV export
- contact submissions CSV export
- unit tests for captcha, contact validation, and interest form validation
- responsive UI

Not included

- payments
- multi seller
- chat
- notifications
- bidding engine
- email sending
- analytics
- ratings
- comments
- favorites
